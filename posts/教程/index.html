hugo+GitHub page 搭建个人博客的使用教程。

初始环境配置

1、hugo插件安装

（1）在hugo下载中下载对应操作系统版本的hugo软件到本地，解压；

（2）将解压后的路径添加进环境变量；

（3）在要保存博客文件的目录下打开命令行，执行 ：

hugo new dreamblog

dreamblog就是生成的博客根目录 （4）测试该博客是否成功生成，继续命令行执行

hugo server

浏览器打开localhost:1313/，如果显示博客内容，则说明成功。 （5）从github上下载博客完整源代码，替换以下标红文件

2、github配置（这里使用桌面版）

（1）下载GitHub桌面版软件，安装，登录账号； （2）克隆项目到本地

记住这个目录

thinker1214.github.io

后面生成的html文件就放在这个里面。

写博文、上传到github服务器

1、准备博文

在下面的路径中创建后缀为md的文件即可，

文件内容的开始格式为：

---
title: 【hugo+GitHub page 搭建个人博客的使用教程】
date: 【2020-01-10T21:27:33+08:00】
lastmod: 【2020-01-10T21:27:33+08:00】
author: 【IrvingBei】
categories:【 ["教程"]】
tags:【 ["hugo"]】
---
【这里是一篇博文的摘要，一般是正文的开始50个字】
<!--more-->
【这里是一篇文章的正文】

上面格式中【】部分进行修改即可。 各个字段的解释：

author => Your name
cover => Current post's image, for quick start, now you can delete this line, because the theme has been placed some default images for posts.
categories => If you're not ready, place "category1"
tags => If you're not ready, place "tag1"
